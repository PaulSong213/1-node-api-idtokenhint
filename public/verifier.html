How E. Zarate Hospital Verifier works?

E. Zarate Hospital Verifier is built on top of Microsoft Entra Verified ID which is a verifiable credentials solution that enables organizations to issue Verifiable Credentials to their users. These credentials can be used to prove identity claims. The solution is built on Open Standards and is interoperable with other verifiable credential solutions.

    3 Easy Steps
  • 1
    Verifier should visit the Verifier Page to Generate QR Code
  • 2
    Medical Certificate Holder will then scan the QR Code through Microsoft Authenticator App
  • 3
    Verifier will then able to view the Medical Certificate Holder's information
